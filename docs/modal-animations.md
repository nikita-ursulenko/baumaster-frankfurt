# Анимации модального окна карточек услуг

## Обзор
Реализованы плавные анимации для модального окна карточек услуг на главной странице `index.php`.

## Реализованные анимации

### 1. Анимация открытия модального окна
- **Фон (backdrop)**: Плавное появление с размытием (blur effect)
- **Контент**: Анимация масштабирования и перемещения снизу вверх
- **Задержка**: 0.1s для контента после появления фона

### 2. Анимация элементов внутри модального окна
- **Заголовок**: Появление сверху с задержкой 0.2s
- **Изображения**: Масштабирование с эффектом zoom-in
- **Контент**: Последовательное появление элементов с задержками
- **Список функций**: Анимация слева направо с задержками
- **Галерея**: Поочередное появление изображений

### 3. Интерактивные эффекты
- **Кнопка закрытия**: Поворот на 90° при наведении
- **Изображения**: Масштабирование при наведении
- **Элементы списка**: Подсветка и смещение при наведении
- **Заголовки**: Градиентный текст и смещение при наведении

### 4. Анимация закрытия
- **Плавное исчезновение**: 0.4s с cubic-bezier easing
- **Синхронизация**: Все элементы анимируются одновременно

## CSS классы

### Основные классы
- `.show` - активирует анимации открытия
- `.loading` - анимация загрузки
- `.success` - анимация успеха
- `.error` - анимация ошибки

### Анимации
- `cubic-bezier(0.4, 0, 0.2, 1)` - основная easing функция
- `backdrop-filter: blur(8px)` - размытие фона
- `transform` - для перемещений и масштабирования

## JavaScript функции

### openServiceModal(serviceId)
1. Находит данные услуги
2. Создает HTML контент
3. Показывает модальное окно
4. Добавляет класс `.show` для анимации

### closeServiceModal()
1. Убирает класс `.show`
2. Ждет завершения анимации (400ms)
3. Скрывает модальное окно
